# Vila Galé + Litoral Verde Landing Page

Landing page promocional para resorts all-inclusive no Brasil, focada no público argentino.

## Tecnologias

- HTML5 Semântico
- CSS3 Moderno (Variáveis, Flexbox, Grid)
- JavaScript Vanilla (Sem dependências)

## Como visualizar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

## Funcionalidades

- Design responsivo (Mobile/Desktop)
- Animações de scroll (Intersection Observer)
- FAQ interativo
- Botão flutuante de WhatsApp
- Otimização de performance e SEO

## Estrutura de Arquivos

- `index.html`: Estrutura da página
- `styles.css`: Estilização completa
- `script.js`: Interatividade e animações
- `images/`: (Se houver pasta, ou arquivos na raiz)
